# Calculadora de IMC 🧮

Una **herramienta web interactiva** que permite calcular el **Índice de Masa Corporal (IMC)** en función del peso y la altura del usuario, y entrega:

- 📊 Estado de salud según la **escala completa de IMC**  
- 🧠 **Categoría detallada** (Delgadez, Normal, Sobrepeso, Obesidad I, II, III)  
- 📍 **Peso ideal promedio y rango recomendado**  
- 💡 Mensajes explicativos y colores por categoría

Este proyecto está hecho con **HTML, CSS y JavaScript**, ideal para practicar lógica, DOM y manipulación dinámica.
